== Description ==

If you want to keep real-time track of your post views / hit by user then you can use this plugins.

This plugins will gives stat for each post views by user. You will be able to see the post views count in the all post view page in WordPress admin panel, also you can use the widget to display the most popular post (based on most views) in the sidebar ares.

== Installation ==

1. Download the plugin and extract the files
2. Upload "cn-post-views-stats" to your wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Check the "Post Views Stats" Tab created by this plugins to see stats

== Frequently Asked Questions ==

= Question : How to use this plugins =

Ans : After install this plugins, you don't need to do anything, this plugins will track post views by user automatically. A new column "Views" will be visible for post listing in the admin panel, where you can see the count for each post.
